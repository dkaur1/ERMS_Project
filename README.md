# ERMS_Project
System Flow Overview

1. User runs main_menu.py
2. System prompts login()
3. After login success, and if user is admin or manager, main_menu(user) shows
4. User selects module (Employee, Dept, Project...)
5. Each module has its own submenu
6. Operations are executed via backend functions
7. Errors are caught and displayed
8. Loop continues until user chooses Exit
